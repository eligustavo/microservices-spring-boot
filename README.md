<h2>Microservice for Movies and catalog</h2>

* Java 11 or greater
* Maven
* SpringBoot 2.3.8.RELEASE or greater
* Intellij IDEA


```shell script

# server ports

# eureka 
http://localhost:8761/

# catalog
http://localhost:8081/catalog

# movies
http://localhost:8082/movies

# ratings
http://localhost:8083/ratingsdata

```
